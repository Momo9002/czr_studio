# CZR Studio — Content DNA

> Read by: Instagram scheduling, campaign scripts, portfolio system

---

## Content Pillars

| Pillar | Ratio | Mode | Purpose |
|---|---|---|---|
| **Portfolio — Light** | 30% | Cream | Show the *result*. The client's world, alive. |
| **Portfolio — Dark** | 15% | Dark | Show the *craft*. The process, the precision. |
| **The Knight** | 20% | Dark | Brand philosophy made visual. Not mascot — manifesto. |
| **Behind the Build** | 20% | Alternating | Process, decisions, the "why" — carousels and reels. |
| **Axiom** | 15% | Alternating | One sentence that takes a position. Not motivation — conviction. |

---

## Soul Rules per Pillar

### Portfolio Posts
Every portfolio post must answer: **what makes this project irreplaceable?**

- Never describe what a site IS (e-commerce, portfolio, landing page). Describe what it DOES or REFUSES.
- Never use the same adjective twice across portfolio posts.
- Find the one constraint, obsession, or philosophical stance that defines the client.
- The caption should make someone who doesn't care about web design stop scrolling.

### Knight Posts
The Knight is not a mascot. It's a position.

- Each Knight post should express a refusal, a decision, or a philosophy.
- Never motivational quotes. Always specific to how CZR operates.
- Test: if a generic design studio could post this, throw it away.

### Behind the Build
Show the thinking, not the doing.

- Not "here's a screenshot." Instead: "here's the decision that made this screenshot inevitable."
- Carousel format: build tension across slides, reveal at the end.
- Always end with the finished result or a metric.

### Axiom
One sentence. Maximum 10 words. Takes a side.

- Bad: "Great design matters." ← says nothing
- Good: "We removed the kickoff call." ← says everything
- Bad: "Simplicity is key." ← bumper sticker
- Good: "Twelve commissions per year. No more." ← conviction

## Grid Pattern (Instagram 3-column)

```
[Dark: Knight]          [Light: Portfolio A]   [Dark: Portfolio B]
[Light: Behind Build]   [Dark: Axiom]          [Light: Portfolio C]
[Dark: Knight]          [Light: Portfolio D]   [Dark: Behind Build]
```

**Rules:**
- Knight appears every 6 posts (1 per every 2 rows)
- Never 3 dark posts in a row
- Never 3 light posts in a row
- No two portfolio posts from the same industry adjacent to each other
- Each portfolio post must feel visually distinct from its neighbours (different industry mood)

## Reels Strategy (no face)

### Priority order (highest to lowest impact)

1. **Screen recording of finished site** — scroll through client site, cinematic audio. Make it feel like a film trailer for a website.
2. **Before/after** — split screen old vs CZR-built site. The contrast does the selling.
3. **Knight reveal** — slow camera push to Knight on desk, laptop reveals site. Atmospheric.
4. **Design timelapse** — code appearing at 8x speed, trending audio. Show the density of craft.
5. **Decision reveal** — text appears word by word: the one decision that defined the project.

**Audio:** Always instrumental. No lyrics. Ryuichi Sakamoto, Nils Frahm, Hans Zimmer piano. Or trending lo-fi/ambient.

**Never:** text-on-black slideshows compiled into video. Never talking-head. Never screen-share tutorials.

## Post Deletion Criteria

Remove a post if:
- It has 0 engagement after 14 days
- It breaks the visual rhythm (wrong mode, inconsistent style)
- It was created before the Knight DNA was adopted
- It contains text-on-black graphics
- It could have been posted by any other design studio (the soul test)
