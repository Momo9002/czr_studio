# CZR Campaign Contract

> Source: `dna/identity.json`, `dna/voice.md`, `dna/visual.md`
> Apply this before creating any Meta/Instagram ad.

---

## Image Requirements

| Mode | Background | Device | Props |
|---|---|---|---|
| Dark ad | `#000000` / nero marble | MacBook | Knight piece, espresso, pen |
| Light ad | `#F7F4EF` / linen | iPhone/iPad | Ceramic cup, dried grass, books |

- Minimum 1080×1080px
- File: PNG, under 8MB
- Always shows real CZR client work on device screen
- No text overlays on the image

## Copy Rules

Primary text (one line max):
```
Brief in minutes. Live in 48 hours.
```
Or:
```
Your site. 48 hours. No compromise.
```

CTA button: `Learn More` → links to `https://czr.studio`

Caption: use `voice.md` Portfolio template

## Targeting Rules

- Geo: UAE first, then US/UK/FR for scaling
- Age: 28–45
- Interests: Web design, Entrepreneurship, Brand management
- Platform: Instagram Feed + Explore + Reels
- Budget: Start $10/day, +20% max per week, never reset learning phase

## Hashtags

Use `identity.json → hashtags` array. Always the same 10.

## Kill Criteria (pause ad if)

- CTR < 0.8% after 7 days
- CPC > $3.00
- 0 WhatsApp messages from campaign after 14 days
