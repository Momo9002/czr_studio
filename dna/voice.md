# CZR Studio — Voice DNA

> Read by: agents, campaign scripts, copywriters, social posts

---

## The Voice Model

**Vogue editor × Hermès copywriter × SpaceX mission brief × Apple product reveal.**

Vogue: editorial authority. The page itself is a statement. Every word is typeset, not typed. Fearless curation — what you exclude defines you more than what you include.
Hermès: quiet confidence. Never explains. Lets silence do the work. A single object in frame says more than a catalogue.
SpaceX: mission-critical precision. No decoration. Data over adjectives. Countdown mentality — there is a deadline and it will be met.
Apple: product reveals, not descriptions. Show the thing. Let it breathe. Obsessive polish. The interface disappears — the experience remains.

Together: luxury that means business, narrated with absolute intention, presented with editorial conviction.

---

## Core Rules

1. **Never sell. State.** — "We deliver in 48 hours." Not "We can deliver in 48 hours!"
2. **Period endings.** Every sentence. No exclamation marks. Ever.
3. **Short sentences.** One idea per sentence. Maximum 12 words.
4. **Never explain. Imply.** — "From brief to live." Not "Our process takes you from the initial briefing call all the way to a live website."
5. **"We do" not "we can."** Ability is assumed. Action is stated.
6. **No emojis.** In copy, captions, or agent messages.
7. **No buzzwords.** See banned list below.
8. **Every word earns its place.** If removing a word doesn't break the sentence, remove it.
9. **Specificity over abstraction.** "14 silhouettes" over "a collection." "Board-formed concrete" over "raw materials."
10. **Each project has a soul.** Never describe two projects the same way.
11. **Editorial, not promotional.** Write as if curating a magazine feature, not selling a service.
12. **White space IS content.** Silence between sentences matters as much as the words. Every pause is designed.

## Banned Words

```
amazing, excited, leverage, synergy, innovative, cutting-edge,
game-changing, disruptive, seamless, holistic, bespoke (use "made"),
solution (use "site" or "work"), deliverable, onboarding (use "start"),
elevate, empower, ecosystem, scalable, next-gen, unmatched, world-class,
best-in-class, state-of-the-art, robust, streamline, optimize,
unlock, supercharge, transform, revolutionize, passionate, dedicated,
high-quality, top-notch, take it to the next level
```

## Caption Architecture

### Portfolio Post — Find the Soul

Do NOT default to a template. Each post must answer: **what makes this project irreplaceable?**

Pattern (adapt, never copy verbatim):
```
[Name in caps].

[The ONE detail that defines this project — not what it IS but what it DOES or REFUSES to do].

[Optional: a surprising fact, a constraint, or a result].

→ Link in bio

#CZRStudio #HauteCoutureDigital [2-3 industry tags]
```

**Examples of soul-finding:**
- Bad: "Fashion house website. E-commerce. Delivered in 48h." ← describes every fashion site
- Good: "ELARA. Fourteen silhouettes. Appointment-only. No cart — you request a private viewing." ← describes ONLY this one
- Bad: "Architecture studio site. Clean design." ← meaningless
- Good: "MATIÈRE. Twelve commissions per year. No decoration. The structure justifies itself or it's removed." ← you feel the philosophy

### Knight / Brand Post
```
[A statement so specific it could only come from CZR].

[Not a motivational quote. A position. A refusal. A decision.]

→ czr.studio
```

**Examples:**
- Bad: "Great design takes time." ← could be anyone
- Good: "We removed the kickoff call. The discovery phase. The approval chain. What remained was the work." ← only CZR says this
- Good: "Not faster because we cut corners. Faster because we removed the corners." ← conviction

### Process Carousel
```
Slide 1: [A tension — something the audience assumes is true that isn't]
Slide 2: [The conventional approach and why it fails]
Slide 3-4: [What CZR does instead — show, don't tell]
Slide 5: [Proof — screenshot, metric, timeline]
Final: → czr.studio
```

### Agent / WhatsApp Intro


## Tone by Context

| Context | Register | Example |
|---|---|---|
| Instagram captions | Vogue editorial. Typography. Every word typeset. | "One table. Twelve guests. Monaco." |
| WhatsApp (agents) | Hermès concierge. Warm, precise, closes deals. | "Your brief is locked. The team starts now." |
| Website body copy | Apple product page. Clinical. Data-driven. | "Delivered in 48 hours. Six pages. Full e-commerce." |
| Ad copy | SpaceX countdown. One line of tension + CTA. | "Three agencies. Six weeks. We quoted 48 hours." |
| Portfolio captions | The one thing that makes this project irreplaceable. | "72 bottles per edition. Each one numbered." |
| LinkedIn | Founder voice. Contrarian observation. Story, not pitch. | "I built six sites in two weeks. Here's what I eliminated." |

## The CZR Sign-off

Always one of:
- `→ czr.studio`
- `DM to start.`
- `Brief us.`

Never: "Contact us today!", "Click the link in bio!", "Drop a comment below!"

---

## The Immersion Test

Before publishing any piece of copy, ask:
1. Could another studio have written this? If yes → rewrite.
2. Does it contain a specific detail only this project has? If no → find it.
3. Would you remember this caption tomorrow? If no → sharpen.
4. Does reading it make you feel something? If no → cut until it does.
5. Would Vogue print this? If no → elevate the editorial quality.
