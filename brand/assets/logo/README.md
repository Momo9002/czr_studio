# CZR Studio — Logo Assets

> Real brand logos and mark files.

---

## Primary Logo

The CZR. wordmark with Mondrian color rays radiating from behind the text on a pure black background. Used as the OG image and primary brand mark.

**File:** `og_image.png` (in this folder — also deployed at `images/og_image.png`)

### Logo Elements

| Element | Description |
|---|---|
| **Wordmark** | `CZR.` — Syne 800, white, tight tracking |
| **Tagline** | `HAUTE COUTURE DIGITAL · DELIVERED IN 48H` — Manrope, uppercase, spaced |
| **Rays** | Mondrian-colored beams (red, yellow, blue + subtle grays) radiating outward |
| **Background** | Pure black `#000000` |
| **URL** | `czr.studio` bottom-right, small |

### Website Favicon

The website uses a `◈` (diamond with cross) character as a data URI SVG:

```html
<link rel="icon" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='.9em' font-size='90'>◈</text></svg>">
```

### Website Wordmark (Header)

In the header/nav, the logo is simply the text `CZR.` in Syne 800 at `1.4rem`, with a `--m-red` underline that animates on hover.
