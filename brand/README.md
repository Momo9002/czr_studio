# CZR Studio — Brand Kit

> The single source of truth for CZR Studio's visual identity, voice, and assets.
> Designed to be scraped by agents, swarms, and collaborators.

## Folder Structure

```
brand/
├── README.md              ← You are here
├── identity.md            ← Brand DNA: mission, values, personality, positioning
├── design-system.md       ← Colors, typography, spacing, components
├── voice-and-tone.md      ← Copywriting rules, vocabulary, tone guidelines
├── assets/
│   ├── logo/              ← Logo variants (SVG, PNG, favicon)
│   ├── colors/            ← Color palette swatches
│   ├── typography/        ← Font specimens & usage rules
│   └── social/            ← Social media templates, profile assets
└── guidelines/
    ├── usage-rules.md     ← Do's and don'ts for brand application
    └── social-media.md    ← Platform-specific posting guidelines
```
