Studio Website

The Mandate

Portfolio site for a boutique architecture studio. Work speaks. Site should not interrupt.

Client

NA Studio

Location

Paris

Focus

Editorial Restraint

The Method

Grid-based editorial layout. Full-bleed project images. Restraint as design language.

The design is intentionally recessive. We built a structured, minimalist grid that allows the architectural photography to dominate. The user interface is discreet, with typography and navigation elements designed to support, never compete with, the portfolio pieces. The entire experience is built on the principle of restraint, ensuring nothing distracts from the studio's work.
