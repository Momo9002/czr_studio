Architecture

Studio Website

The Challenge

The work speaks. The site must not interrupt.

For a boutique architecture studio, the portfolio is the argument. The digital presence needed to be a silent, structural frame that presented the work with confidence and clarity, getting out of the way to let the projects command attention.

Client
NA Studio
Location
Paris

The Build

Restraint as a design language.

A grid-based editorial layout was established, using full-bleed project images to create an immersive experience. The design language was one of pure restraint, focusing on typography, structure, and space to create a gallery-like environment.

  • Website
  • Portfolio system
  • Press kit

The Result

Featured in Dezeen.

The project's quiet confidence earned a feature in Dezeen, the industry's leading publication. This exposure led directly to three new studio commissions within two months of launch.
