Studio Portfolio

The Mandate

Photography portfolio where the work is the product. Site must not compete with the images.

Client

Studio Kove

Location

London

Tagline

The image, uninterrupted.

The Method

Full-bleed grid. Lightbox reveals. Booking inquiry form. Minimal type.

The design is an exercise in absence. We constructed a simple, full-bleed grid that puts the photography first. There are no unnecessary visual elements. Clicking an image reveals a full-screen lightbox for an uninterrupted viewing experience. The only typographic elements are essential captions and a discreet booking form. The site is a quiet frame for the work.

  • Portfolio site
  • Booking system
  • Image hosting

The Result

Now fully booked six months in advance. Client list includes two fashion weeks.
